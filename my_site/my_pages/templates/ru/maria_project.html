Удобная работа с БД

Отображение базы данных в удобном для пользователя html-шаблоне. Все изображения, тексты на сайте вынесены в БД, что дает возможность изменять практически все на сайте не обращаясь к программисту.

Мною выполнено:
- сайт выполнен в двух вариантах, с базой MySQL / PostgreSQL.
- создание панели доступа, для возможности изменения контента самим клиентом.
- настройка доступа к панели изменения данных.
- запросы к БД для вывода информации.

Docker-compose

Развертывание и запуск веб-приложения выполнено в его рабочей среде через Docker на сервере Ubuntu, хостинге VDSINA.

Мною выполнено:
- подготовлен образ сайта в dockerfile с последующим pull в docker hub.
- на удаленном сервере Ubuntu через командную строку поднят Docker, скачены образы c docker hub.
- создан файл docker-compose.yml через командную строку и редактор VIM.
- запуск сайта на удаленном серевер в DOCKER.
